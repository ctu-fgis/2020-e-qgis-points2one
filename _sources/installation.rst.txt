
Installation
==============
You can download plugin from `Github
<https://github.com/ctu-fgis/2020-e-qgis-points2one.git>`__  as a ZIP file.
 
The plugin can be installed by selecting in the top panel: **Plugins â†’ Manage and Install Plugins**
You have to click on the item **Install from ZIP** then select the path to the folder with ZIP and click on the **Install Plugin**.

.. image:: images/installation.png

QGIS displays a warning message. If you want to continue select **Yes**.

.. image:: images/warning.png
   
Now you can see a new icon in the QGIS toolbar.
    
.. image:: images/icon.png
